# 🏨 Hotel & Resort Website

A modern, responsive **Hotel & Resort Website** built using **HTML, CSS, and Bootstrap 5**. This project showcases a luxury hotel experience with elegant UI design, smooth animations, and multiple sections like rooms, dining, spa, and events.

## ✨ Features

* 🔹 Fully responsive design (mobile, tablet, desktop)
* 🔹 Elegant navigation bar with dropdown menus
* 🔹 Image carousel with smooth transitions
* 🔹 About section with animations
* 🔹 Rooms & Suites showcase
* 🔹 Dining and service sections
* 🔹 Spa & Wellness information
* 🔹 Events & experiences section
* 🔹 Interactive cards with hover effects
* 🔹 Stylish footer with contact details & social links

## 🚀 How to Run the Project

1. Download or clone the repository

   ```
   git clone https://github.com/your-username/hotel-resort.git
   ```

2. Open the project folder

3. Run the project by opening:

   ```
   index.html
   ```

---

## 🎨 Customization

You can easily customize:

* Colors → Modify in `style.css`
* Images → Replace inside `/asset` folder
* Content → Edit text directly in `index.html`
* Layout → Use Bootstrap grid system
